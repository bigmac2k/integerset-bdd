    Iteratively constructs a CBDD from a path to a terminal represented as a list of Booleans, two CBDD subtrees, and a terminal CBDD. If the path is empty, terminal is returned. If head is true, a Node is constructed where set is the tree constructed from the remaining list, unset is uset argument. If head is false, set and unset are swapped.
